# 개요
## github란
git을 활용하기 좋게 만든 사이트인데 보통 git하면 github가 개발자들은 생각하지만   
영화나 미디어 분야에서는 gitlab을 많이사용하기도해서  
git하면은 github라 생각하는것은 좋은생각이 아니라 생각한다.
