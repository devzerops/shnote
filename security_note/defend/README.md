# 개요
`security_note/defend/`는 공격 벡터에 대응하기 위한 방어 전략, 솔루션 배치 사례, 운영 체크리스트를 모읍니다. EDR/NDR, NAC, 윈도우 하드닝 등 실무 대응 자료와 연계됩니다.

## 디렉터리 구조
- `ELAM.md`: Windows Early Launch Anti-Malware 개요 및 부팅 보호.
- `NAC.md`: NAC 정책 설계, 솔루션 비교.
- (추가 예정) `SOAR.md`: 자동화 플레이북 및 대응 사례.
