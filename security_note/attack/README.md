# 개요
`security_note/attack/`은 공격 기법, 취약점 악용 흐름, 위협 분류를 정리한 컬렉션입니다. 실무 대응을 위해 공격 표면, 탐지 지표(IOC/IOA), 방어 관점 참고 사항을 함께 기록합니다.

## 디렉터리 구조
- `basic/`: 사회공학, 계정 탈취 등 범용 전술.
- `DNS/`: DNS 기반 위협(DNS 캐시 중독, 스푸핑 등).
- `dos/`: DoS/DDoS 기법과 변형 공격.
- `infra/`, `os/`: 하드웨어·OS 취약점 및 사이드채널.
- `web/`: 웹 취약점 및 공격 시나리오.
