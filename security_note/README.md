# 개요
`security_note/`는 보안 기술, 위협 대응 전략, 솔루션 비교 자료를 수집한 공간입니다. 침해사고 대응, 인증·암호화, 네트워크 방어 등 실무 사례 중심의 문서를 모아두고, 참고 링크와 도구 사용 팁을 함께 기록합니다.

## TL;DR
- `디렉터리 구조`로 공격(`attack/`)과 방어(`defend/`) 흐름을 먼저 파악하고, 대표 문서 링크에서 최신 자료를 확인하세요.
- 침해대응 준비 시 `defend/NAC.md`와 `drawio/` 다이어그램을 함께 읽어 대응 순서를 익힙니다.
- 문서 보강 전 `docs/document_inventory.md`의 상태 메모와 TODO를 확인해 중복 작업을 피하세요.

## 디렉터리 구조
- `attack/`: 공격 기법, 침투 벡터, IOC 정리.
- `defend/`: 방어 전략, 대응 시나리오, 운영 가이드.
- `encryption/`: 암호화 기술, 인증 인프라 관련 메모.
- `tool_principle/`: 보안 솔루션 작동 원리 및 아키텍처.
- `quantum_cryptography_수정필요.md`: 양자암호 초안, 사례 보강 예정.
