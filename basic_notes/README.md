# 개요
`basic_notes/`는 범용 컴퓨팅 이론과 자격 시험 대비 내용을 정리하는 공간입니다. 하위 디렉터리는 주제별로 분리하며, 각 문서는 동일한 템플릿(개요→핵심 개념→실무 포인트→TODO→참고 자료)을 적용합니다.

## TL;DR
- 3단계로 읽으세요: `디렉터리 구조`에서 주제를 고르고 → 해당 폴더 README 또는 대표 문서 → 세부 문서 TODO 확인.
- 메타데이터(`마지막 업데이트`, `상태`)로 최신성을 먼저 판단하고, 내용이 비어 있으면 TODO에 코멘트를 남깁니다.
- 시험 대비가 목적이면 `대표 문서 빠른 링크`와 각 문서의 `실무/시험 포인트`를 우선 확인하세요.

## 디렉터리 구조
- `algorithme/`: 정렬, 탐색, 그래프 등 알고리즘 개요.
- `architecture/`: 컴퓨터 구조·명령어 집합·파이프라인.
- `data/`: 통계 및 데이터 표현 기초.
- `DB/`: 트랜잭션, SQL, 정규화 등 데이터베이스 이론.
- `extension/`: 파일 포맷 및 확장자 메모.
- `math/`: 이산수학·확률 등 기초 수학.
- `network/`: OSI 계층, 라우팅, 프로토콜 요약.
- `server/`: 서버 구성, 가용성 아키텍처.
- `simple_grammar/`: 마크다운·정규식 등 문법 기초.
- `web/`: 웹 구조, HTTP/HTTPS, 상태 코드.
