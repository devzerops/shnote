# 개요
## join이란

* 내부 조진 (INNER JOIN)
    1. 교차 조인 (CROSS JOIN  - CARTESIN JOIN)
    2. 등가/동등/동일 조인(EQUI JOIN)
    3. 비등가 조인(NON-EQUI JOIN)
    4. 자연 조인 (NATURAL JOIN)

* 외부 조인 (OUTER JOIN)
    1.완전 외부 조인 (FULL OUTER JOIN)
    2.왼쪽 (LEFT OUTER)
    3.오른쪽 (RIGHT OUTER)

    4. 셀프 조인 (SELF JOIN)
    5. 안티 조인 (ANTI JOIN)
    6. 세미 조인 (SEMI JOIN)
