# 개요
* 802.1 - 전체의 구성
* 802.2 - 논리 링크 제어(LLC)
* 802.3 - CSMA/CD 방식
* 802.4 - 토큰 버스 방식
* 802.5 - 토큰 링 방식
* 802.6 - 도시형 통신망(MAN),이중 버스 통신망(DQDB)
* 802.11 - 무선 LAN
* 802.15 - 블루투스,WPAN
