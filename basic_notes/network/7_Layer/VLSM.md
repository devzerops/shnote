# 개요
## VLSM(Variable Length Subnet mask, rkqus rlfdldml tjqmspt aktmzld)
서브네팅을 여러 번 반복하여 네트워크를 크기가 다른 여러 서브네트워크 계층으로 구분하는 기법입니다.  

이를 통해 기관은 네트워크의 실제 요구 사항에 맞게 서브넷의 크기를 조정할수 있습니다.(IP를 효율적으로 할당하여 활용하는 방법)
