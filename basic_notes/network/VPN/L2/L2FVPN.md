# 개요
## L2F(Layer 2 Forwarding Protocol)
* 시스코(Cisco Systems)사에서 제안된 프로토콜로서 NAX 개시 VPN형이기 떄문에 사용자는 별도의 S/W가 필요가 없다.
* 하나의 터널에 여러 개의 연결을 지원하여 다자간 통신이 가능하다.
* 전송 계층 프로토콜로 TCP가 아닌 UDP(User Datagram Protocol)을 사용한다.
